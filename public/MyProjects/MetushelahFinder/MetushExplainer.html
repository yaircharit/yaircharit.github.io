Metushelah Finder Showcase

A Conway's Game of Life research tool that evolves random patterns into the longest-lived and most expansive automata using rank-based genetic selection and mutation.

Overview

Metushelah Finder is implemented with the seagull library and uses two core classes: CAAnalyzer and GeneticAlg. The analyzer runs each candidate through Conway's Game of Life, tracking lifespan, current coverage, peak coverage, and termination state (alive / dead / looper).

The genetic algorithm evaluates each candidate using a fitness score defined as lifespan × peak coverage, then applies rank-based selection and mutation to build the next generation.

  • Example board size: 150 × 150
  • Population size: 10 candidates per generation
  • Max simulation iterations per candidate: 5,000
  • Fitness = lifespan × peak coverage
  • Mutation strategy: flip a random bit in the starter pattern
  • Selection strategy: rank selection with niche-style diversity preservation

Key algorithm decisions

  • Structured evaluation: each candidate is simulated until it dies, enters a loop, or reaches the iteration cap. The analyzer records the exact moment of highest board coverage.
  • Rank selection: stronger candidates receive proportionally more offspring, but the algorithm prevents total domination by using a floor-based count and keeping weaker candidates in the ranking process.
  • Mutation operator: offspring are generated by flipping a random cell in the parent pattern, which explores nearby starting configurations without discarding the parent’s structure entirely.
  • Diversity preservation: the algorithm injects new random candidates each generation and uses a niche-inspired selection step to avoid premature convergence.
  • Bounded runtime: a 5,000-iteration cap keeps simulation time manageable while still allowing long-lived automata to emerge.

What this demonstrates

Metushelah Finder showcases how a genetic search can find complex emergent behavior from simple rules:

  • Small random starter patterns can grow into oscillators or stable loops.
  • Fitness that combines lifespan and coverage encourages both durability and expansion.
  • Rank selection and mutation preserve exploration while letting the best patterns dominate.
